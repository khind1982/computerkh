This utility was developed to provide the editor with a quick way to isolate and inspect certain fields in XML data.

To use this utility simply enter a username(this will be used for logging errors/queries), the path to the data and tag(s) that you would like to validate.

Issues that need to be looked at are spelling mistakes, garbles text and uniformity in fieds where uniformity is expected, eg. dates (numdate, originaldata).

 

 

*** Warning! / Disclaimer ***

  • This utility was developed using Chrome. It *might* work on other browsers but if it doesn't you're on your own.
  • Multiple tags that share the same name won't work!
  • Please, please, please check the file permissions before you try to edit them. File permissions should be set to 777!
